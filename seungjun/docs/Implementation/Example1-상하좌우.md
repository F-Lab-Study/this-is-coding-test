# 상하좌우

## 문제

크기가 N*N인 좌표에서 L, R, U, D(상하좌우) 입력을 한만큼 (1, 1)에서 시작해서 좌표 이동을 한다.<br>
단, 좌표 크기를 벗어나는 경우에는 이동을 하지 않는다.

### Input

* 첫째 줄에 공간의 크기를 나타내는 N(1 <= N <= 100)이 주어진다.
* 둘째 줄에 여행가 A가 이동할 계획서 내용이 주어진다.(1 <= 이동횟수 < =100)

```
5
R R R U D D
```

### Output

* 첫째 줄에 도착할 지점의 좌표 (X, Y)를 출력한다.

```
3 4
```
